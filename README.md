# Sistema de Gestión de Inventario con Árbol Binario de Búsqueda (ABB) en Python

## Descripción del proyecto

Este proyecto implementa un **Sistema de Gestión de Inventario** para pequeñas tiendas o negocios locales. Fue desarrollado en **Python** sin frameworks externos, utilizando estructuras de datos clásicas como un **Árbol Binario de Búsqueda (ABB)** para almacenar productos por código y permitir búsquedas rápidas, y un **diccionario (hash table)** para agrupar productos por categorías.

El sistema corre en consola y permite registrar productos, buscar rápidamente por código, listar por categorías, eliminar productos y mostrar el inventario completo.

---

## Requisitos

- Python 3.6 o superior
- Consola o terminal para ejecutar el programa
